### 游戏简介

1940年夏秋之交，纳释德国发动了对英国的全面空袭，史称不列颠空战。这是人类历史上首次以空军   
为主导的大规模战役，英国皇家空军以少胜多，用血肉之躯捍卫了国家主权。

#### 游戏玩法说明
**飞行操控**   
方向控制：通过方向键（一←↓→）整战机航向，躲避敌机，闪避机动      
**火力对抗**   
主炮射击：默认空格键发射机炮（伤害1），弹道为直线   
**敌机威胁**   
1型战斗机（JunkersJu87）：机动性高，擅长俯冲突袭，血量较低（1点）。   
2型轰炸机（HeinkelHe111）：移动缓慢但装甲厚重（3点血量）   
**生存系统**   
血条机制：初始3点生命值，被击中后闪烁进入无敌状态（持续1秒），期间可无视伤害但无法攻击。   
紧急迫降：生命归零时触发坠机动画，按M键可返回主菜单重整旗鼓。   
**特殊技能**   
每300分会获得一个技能，按v发射，按b爆炸。   
**难度梯度**   
分数大于等于100开始改变：   
1. 每10分多1个敌机，上限是60
2. 每200分‘1’型敌机速度加1，上限是5
3. 每1000分单次敌机生成加1
4. 每50分射击间隔缩短20，最少为60
5. 每300分加1点技能，每多2000分额外多加1点
(*其中设置了防止难度陡增的机制，详见源代码*)
