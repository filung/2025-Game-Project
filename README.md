# 2025-Game-Project
这是scu一个学生小组的C语言期末游戏项目，本游戏为改版的小飞机，制作比较粗糙，如果有其他同学看到了可以参考，但是拒绝抄袭，包含相关的简介ppt以及项目报告（真实信息已经删去）
---
资料说明：

1. 安装包 mysetup.exe
2. 项目报告汇总
    - 第组-游戏项目-报告.doc /word/
    - 第组-游戏项目汇报PPT.pptx /PPT/
3. 游戏程序(源代码cpp文件）
4. 相关的资源文件(resource文件夹)
5. 相关的说明(本文档)
---
相关的运行源代码说明

- 建议在VS2026下运行源代码
- 需要安装easyx图形库
- 相关的设置请看相关的中文输出问题解决方案的markdown的文档，否则中文输出输入会乱码
- resource文件夹要放在main.cpp的统一目录下
- 只是想玩的话直接下载安装包setup
